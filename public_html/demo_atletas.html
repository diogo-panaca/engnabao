Atletas Olímpicos Portugueses

Esta página demonstra a programação de uma aplicação web simples. A interface presente nesta página permite ao utilizador consultar louvores obtidos por três atletas portugueses em várias competições internacionais.

A página contém um formulário em que se pode escolher os filtros a utilizar. Esta é uma versão inicial em que é obrigatório escolher um e apenas um atleta. Para este efeito, marca-se o botão de rádio correspondente.

A página foi desenvolvida com tecnologias HTML5 e faz chamadas AJAX, em que envia dados relevantes em formato JSON para o servidor. Lá, os dados tratados por um script PHP que consulta a base de dados, converte informação obtida no formato JSON. Depois, devolve os dados ao cliente, que apresenta numa tabela HTML, debaixo do formulário.
