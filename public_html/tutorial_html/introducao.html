Documento HTML

Um documento HTML é um ficheiro de texto que descreve o conteúdo de uma página da internet. Os navegadores (browsers) lêem este tipo de documento e desenham a página que lá está descrita.

Pode-se utilizar um editor de texto para editar o código HTML5 e abri-lo num navegador moderno. Numa primeira fase, basta abrir o ficheiro escrevendo o caminho local na barra de endereço; futuramente pode ser necessário um servidor.

Elementos

As páginas de internet são compostas por elementos que definem a sua estrutura. Num documento HTML, descreve-se esta estrutura com marcas (é por isso que se chama uma lingugem de demarcação - HyperText Markup Language).

Exemplo de um elemento com conteúdo:

<p>Lorem ipsum dolor sit amet</p>

Exemplo de um elemento sem conteúdo (mas que podiar ter):

<p> </p>

Exemplo de um elemento vazio:

<img>

Estrutura de um documento HTML5

<!DOCTYPE html>
<html>
  <head> </head>
  <body> </body>
</html>
  1. O elemento DOCTYPE indica a variante de HTML utilizada. Desta forma distingue-se de outros formatos, como XML.
  2. O elemento html possui o conteúdo do documento.
  3. O elemento head é o cabeçalho do documento HTML. Possui informação sobre o próprio documento ou da página, como o título no separador.
  4. A marca body é o corpo do documento HTML. A mesma representa o conteúdo e a estrutura da página web - texto, imagens, tabelas, etc.
