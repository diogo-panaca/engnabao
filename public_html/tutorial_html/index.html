Documento HTML

Um documento HTML é um ficheiro de texto que descreve o conteúdo de uma página da internet. Os navegadores (browsers) lêem este tipo de documento e desenham a página que lá está descrita.

Pode-se utilizar um editor de texto para editar o código HTML5 e abri-lo num navegador moderno. Numa primeira fase, basta abrir o ficheiro escrevendo o caminho local na barra de endereço; futuramente pode ser necessário um servidor.

Elementos

As páginas de internet são compostas por elementos que definem a sua estrutura. Num documento HTML, descreve-se esta estrutura com marcas (é por isso que se chama uma lingugem de demarcação - HyperText Markup Language).

Exemplo de um elemento com conteúdo:

<p>Lorem ipsum dolor sit amet</p>

Exemplo de um elemento sem conteúdo (mas que podiar ter):

<p> </p>

Elementos vazios são aqueles que não podem ter conteúdo. São constituídos por uma única marca, que se fecha a si própria.

Exemplo de um elemento vazio:

<img/> ou então <img>

Estrutura de um documento HTML5

Este é um dos exemplos de documentos HTML5 mais pequenos. Não possui conteúdo, apenas o "esqueleto": 

    
<!DOCTYPE html>
<html>
  <head> </head>
  <body> </body>
</html>
  1. O elemento DOCTYPE indica a variante de HTML utilizada ― neste caso a indicação "html" refere-se a HTML5. Desta forma distingue-se de outros formatos, como XML.
  2. O elemento html é o elemento raiz, cujo conteúdo são todos os outros elementos.
  3. O elemento head é o cabeçalho do documento HTML. Possui informação sobre o próprio documento ou da página, como o título da página no separador do navegador.
  4. O elemento body é o corpo do documento HTML. Representa o conteúdo e a estrutura da página web ― texto, imagens, tabelas, etc.
